Verbindliche Begriffe. Abweichungen erzeugen Inkonsistenzen zwischen den Chunks,
weil dieselben Begriffe in Item-Namen, Effekten und Ereignistexten wiederkehren.

## Attribute

STR → STÄ · INT → INT · CON → KON · DEX → GES · CHA → CHA

Strength → Stärke · Intelligence → Intelligenz · Constitution → Konstitution ·
Dexterity → Geschicklichkeit · Charisma → Charisma

## Raritäten

Common → Gewöhnlich · Uncommon → Ungewöhnlich · Rare → Selten ·
Very Rare → Sehr selten · Epic → Episch · Legendary → Legendär ·
Mythic / Mythical → Mythisch · Unique → Einzigartig · Celestial → Himmlisch ·
Divine → Göttlich · Platinum → Platin · Bronze → Bronze · Gold → Gold ·
Enchanted → Verzaubert

`Divine / Unique` → `Göttlich / Einzigartig` (Schrägstrich beibehalten)

## Ausrüstungsplätze

head → Kopf · face → Gesicht · neck → Hals · shoulders → Schultern ·
chest → Brust · back → Rücken · hands → Hände · wrists → Handgelenke ·
waist → Taille · legs → Beine · knees → Knie · ankle → Knöchel ·
feet → Füße · toes → Zehen · ring → Ring · piercing → Piercing ·
patch → Aufnäher · trinket → Schmuckstück · weapon → Waffe ·
offhand → Nebenhand · mount → Reittier · costume → Kostüm

## Eigennamen

Figurennamen bleiben unverändert: Carl, Donut, Mordecai, Katia, Samantha,
Bautista, Li Na, Elle, Imani, Louis, Firas, Gustavo, Rend, Signet, Grimaldi …

`Princess Donut` → `Prinzessin Donut`
`Dungeon Crawler Carl` (Serientitel) → unverändert
`Matt Dinniman` (Autor) → unverändert

Gegenstands-, Fähigkeits-, Zauber- und Titelnamen **werden übersetzt** — das ist
die ausdrückliche Vorgabe des Nutzers. Wo ein Name ein Wortspiel ist, geht
Verständlichkeit vor Wörtlichkeit.

## Stil

- Beschreibender Ton, keine Leseransprache.
- Zeitform wie im Original, meist Präsens oder Perfekt.
- Der Ton der Reihe ist derb und ironisch. Derbheit nicht glätten.
- Deutsche Rechtschreibung samt Umlauten und ß. Niemals `ue` für `ü`.
